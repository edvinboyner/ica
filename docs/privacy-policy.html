Integritetspolicy – ICA Prisjämförelse

Senast uppdaterad: mars 2026

Vad extensionen gör

ICA Prisjämförelse är ett Chrome-tillägg som hjälper dig att jämföra priser på din varukorg mellan ICA-butiker med hemleverans i ditt postnummerområde. Extensionen läser din inloggade ICA-session för att hämta din varukorg och prisdata, och jämför sedan totalkostnaden mellan tillgängliga ICA-butiker. All bearbetning sker lokalt i din webbläsare — ingen data skickas till externa servrar.

Vilken data läses och används?

Extensionen läser följande data från din inloggade session på handlaprivatkund.ica.se, uteslutande i syfte att visa prisjämförelsen:

  • Din aktiva varukorg (produkter och kvantiteter)
  • Prisinformation från ICA-butikernas produktkataloger
  • Vilken ICA-butik du för tillfället handlar i

Denna data lämnar aldrig din webbläsare. Den används enbart för att beräkna och visa prisjämförelsen direkt i tillägget.

Vilken data sparas lokalt?

Extensionen sparar följande data i din webbläsares lokala lagring:

  • Ditt postnummer (för att söka efter butiker med hemleverans)
  • Din nuvarande ICA-butiks ID
  • Senaste jämförelseresultat (tillfälligt, rensas när webbläsaren stängs)

Ingen av dessa uppgifter delas med eller överförs till tredjeparter.

Vad delas inte?

  • Din varukorg eller dina köpvanor delas inte med någon
  • Dina inloggningsuppgifter hos ICA lagras inte av extensionen
  • Inga analytics, spårningscookies eller reklamtjänster används
  • Ingen data säljs eller delas med tredjeparter

Behörigheter och varför de krävs

  • storage – Sparar ditt postnummer och butiks-ID lokalt i webbläsaren mellan sessioner.
  • tabs – Identifierar din öppna ICA-flik för att kunna köra prisjämförelsen mot rätt sida.
  • scripting – Kör prisjämförelsen i kontexten av din ICA-flik (same-origin) för att kunna läsa din varukorg och hämta prisdata via ICAs egna API:er.
  • handlaprivatkund.ica.se, handla.ica.se – Tillåter extensionen att kommunicera med ICA:s webbplats för att hämta prisinformation.

Ta bort data

Du kan när som helst radera all sparad data genom att avinstallera extensionen. Chrome rensar då automatiskt all lokal data som extensionen har sparat.

Ändringar i policyn

Om denna integritetspolicy uppdateras kommer den nya versionen att publiceras på denna sida med ett uppdaterat datum.

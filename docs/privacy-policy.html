Integritetspolicy – ICA Prisjämförelse

Senast uppdaterad: mars 2026

Vad extensionen gör

ICA Prisjämförelse är ett Chrome-tillägg som hjälper dig att jämföra priser på din varukorg mellan ICA-butiker med hemleverans i ditt postnummerområde.

Vilken data samlas in?

Extensionen sparar följande data lokalt i din webbläsare:

  • Ditt postnummer (för att söka efter butiker med hemleverans)
  • Din nuvarande ICA-butiks ID (läses från ICA:s webbsida)

Ingen data skickas till externa servrar. All bearbetning sker direkt i din webbläsare. Ingen tredjepart har tillgång till dina uppgifter.

Vilken data delas inte?

  • Din varukorg eller dina köpvanor delas inte med någon
  • Din inloggningsinformation hos ICA lagras inte av extensionen
  • Inga analytics, spårningscookies eller reklamtjänster används

Hur används din ICA-session?

Extensionen använder din befintliga inloggade session på handlaprivatkund.ica.se för att hämta din varukorg och prisinformation från ICAs egna API:er — på samma sätt som ICA:s webbsida själv gör. Dina inloggningsuppgifter läses eller lagras aldrig av extensionen.

Behörigheter

  • storage – Sparar ditt postnummer och butiks-ID lokalt
  • tabs – Hittar din öppna ICA-flik för att köra jämförelsen
  • scripting – Kör prisjämförelsen i din ICA-flik (same-origin)
  • handlaprivatkund.ica.se, handla.ica.se – Hämtar prisdata från ICA

Ta bort data

Du kan när som helst radera all sparad data genom att avinstallera extensionen. Chrome rensar då all lokal data automatiskt.
